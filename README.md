# debugging-sessions
Several utilities in this repository originated from performance tests that focused on execution speed, transaction preparation, and interaction monitoring. The collection gradually expanded as additional tools proved useful during debugging sessions.

The implementation Leverages lightweight components and avoids unnecessary dependencies whenever possible. This approach makes individual examples easier to modify and inspect.

Some modules were created while experimenting with a powering API workflow that simplified communication between independent services and transaction handlers.

Other examples process operations in batches in order to compare execution results and observe how repeated requests behave under different conditions.
